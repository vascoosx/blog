First steps to start exploring face tracking technology

火 11 3月 2014
By Shunichi Vasco Otsuka

In misc.

If you have been interested in computer vision and face tracking in particular checkout FaceTracker and clmtrackr. Also there are some awesome opensource projects that are very fun(FaceSubstitution, FaceSubstitutionEditor).

So to get you started, what might be better than to test it in your own computerand start hacking in a few minutes!

What you need: Git brew * openframeworks

By the way I will be explaining how to install on a Mac OSX, sorry other users.

First let's download openframeworks.

While downloading OF let's also get two special addons: ofxfacetracker ofxCv

Ofxfacetracker can be cloned with git and used right away. However ofxCv is special. Kyle McDonald author of ofxCv is in the middle of a major remake so the master branch has changed so much that it won't work if you clone it. In order touse it download the zip file from github. Also you will need to rename the expanded zip. It will be named ofxCv-master but I think you have to rename it ofxCv. mv ofxCv-master ofxCv

OK, once OF has finished downloading, extract it and place it to the place of your choice. Let's say you have put it in /my/path/to/of. You will now need to move your addons to /my/path/to/of/addons. Just

mv ofxfacetracker ofxCv /my/path/to/of/addons

Now that we have the addons, download FaceSubstitutionEditor. The place where you will need it is in /my/path/to/of/apps

cd /my/path/to/of/apps
git clone http://FaceSubstitutionEditor

Great! Now you have to build it. Nevertheless it's not that obvious as just typing make. You have run the project generator with the "project-generator". Run the project-generator and add the addons. The path of the generator is /my/path/toof/apps/FaceSubstitutionEditor/ and the name of the project is FaceSubstitution. It is important that the name is as I have written here so be careful with typos.

After the project is generated, you can finally build it.

make

If everything goes alright, you need to put data for the face tracking engine. This data is used by FaceTracker to identify face features. You can get a set of this kind of data from the addon ofxfacetracker. Just copy it.

# Assuming you are in /my/path/to/of/apps/FaceSubstitutionEditor/FaceSubstitution
cp -r ../../../addons/ofxfacetracker/lib/model bin/data

That's it. You now only have to run it like this:

make run

OK, I was not telling the truth. After running it you need to drag an image file and a tab separated file (tsv) containing the feature points. You also need to drag the displayed points to their corresponding places in the image file.
