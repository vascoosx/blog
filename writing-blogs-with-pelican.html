Writing blogs with pelican

月 17 2月 2014
By Shunichi Vasco Otsuka

In IT.

Write a blog in a file and save it into directory content. Next run this command, which will generate files into directory "output".

pelican -s pelicanconf.py -o output content

You can write your blog in different categories. To do so, move file into directory mapped to the category inside directory "content".

To push your blog to github,

make html && ghp-import output && git push origin gh-pages
